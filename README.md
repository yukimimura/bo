# Find Book
<img src="home_sample.png" width="400" height="350">

## 概要
読みたい漫画などの書籍を見つけるためのアプリです。<br>
ジャンルやキーワードで条件検索し、5段階評価をすることができます。<br>
その他にもユーザーをフォローし、メッセージのやり取りなどSNSとしての側面もあります。

## デプロイ
