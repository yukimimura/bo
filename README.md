# Find Book

## 概要
読みたい漫画などの書籍を見つけるためのアプリです。<br>
ジャンルやキーワードで条件検索し、5段階評価をすることができます。<br>
その他にもユーザーをフォローし、メッセージのやり取りなどSNSとしての側面もあります。

## デプロイ
https://find-book-0605.herokuapp.com/

## 開発環境
* Ruby 2.5.3
* Ruby on Rails 5.2.4
* Mysql 5.5.62
* javascript
* jQuery
* Bulma
* AmazonS3
